Dwi di minta tolong oleh temannya untuk menghitung perkalian 2 buah matriks, tetapi dwi tidak mengerti bagaimana caranya, dwi berharap ada program yang dapat membantunya untuk menghitung perkalian 2 matriks. Jadi kalian anak IT bantu lah dwi untuk membuat program keinginannya dengan menggunakan fungsi method dan penyimpanan data array.

Input Format
baris_matriks1 [spasi] kolom_matriks1 baris_matriks2 [spasi] kolom_matriks2 isi_data_matriks1 isi_data_matriks2

Constraints
Method array 1 dimensi atau 2 dimensi ukuran matriks dinamis tergantung inputan

Output Format
hasil perkalian matriks dalam bentuk matriks

2 3
3 3

3 3 1
4 0 3

4 1 1 
3 3 4 
2 1 2

22 7 10
